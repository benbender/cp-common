# @benbender/eslint-config

[ESLint](https://eslint.org) configuration to be shared across repositories.

## Usage

**Install**:

```bash
$ yarn add --dev @benbender/eslint-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "eslintConfig": {
      "extends": [
        "@benbender"
      ]
  }
}
```
