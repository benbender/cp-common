# @benbender/tailwind-preset

[Tailwindcss](http://tailwindcss.com/) preset to be shared across repositories.

## Usage

**Install**:

```bash
$ yarn add --dev @benbender/tailwind-preset
```

**Add `tailwind.config.js`**:

```js
module.exports = {
  presets: [require('@benbender/tailwind-preset')],
}
```
