# @benbender/stylelint-config

[Stylelint](https://stylelint.io) configuration to be shared across repositories.

## Usage

**Install**:

```bash
$ yarn add --dev @benbender/stylelint-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "stylelint": {
    "extends": "@benbender/stylelint-config"
  }
}
```
